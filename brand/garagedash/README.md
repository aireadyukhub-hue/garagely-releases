# GarageDash Brand Pack — v1 (6 Jul 2026)

New GarageDash identity, built from `GarageDash_Brand_Guidelines.docx`. All SVGs have text
converted to outlines (Inter), so they render identically everywhere — no fonts needed.

## The mark

Kept the hexagon G, improved: flat-top orientation (reads as a hex nut), solid GarageDash
Orange `#FF6A00`, rounded corners, heavier G — and the G's crossbar now extends into a **dash**,
the literal "Dash" in GarageDash. Alternatives considered are in `mark-options.png`
(A = plain G, no dash) — say the word to switch.

## Colours (from the brand guidelines)

Accent `#FF6A00` · Void Black `#0A0A0A` · Lift Black `#111111` · Smoke `#999999` · Seam `#1E1E1E`

## Files

- `logos/` — mark, full logo (dark + light backgrounds), badge (any background), email logo (560px)
- `icons/` — app icon SVG + PNG at 16–1024, PWA maskable, 180 = apple-touch
- `ads/` — 4 square social ads (1080×1080), cover banner (1500×500), og-image (1200×630).
  SVG sources included — edit copy and re-export any time.
- `REVIEW-SHEET.png` — everything on one page
