# GarageDash Brand Pack — v1 (6 Jul 2026)

New GarageDash identity, built from `GarageDash_Brand_Guidelines.docx`. All SVGs have text
converted to outlines (Inter), so they render identically everywhere — no fonts needed.

## The mark

Kept the hexagon G, improved: flat-top orientation (reads as a hex nut), rounded corners, heavier
G — and the G's crossbar now extends into a **dash**, the literal "Dash" in GarageDash.
Alternatives considered are in `mark-options.png` (A = plain G, no dash) — say the word to switch.

**Colour update (6 Jul 2026, post-launch):** the initial pack shipped with a new orange `#FF6A00`,
but next to the site/app's established accent (`#F4A523`, yellow-gold, from the earlier June 2026
branding rollout) the two read as mismatched oranges. Reverted the mark/logo/icon/ad colour back to
the original `#F4A523` everywhere so there's one consistent brand colour again.

## Colours (from the brand guidelines)

Accent `#F4A523` (yellow-gold — canonical, matches site/app) · Void Black `#0A0A0A` ·
Lift Black `#111111` · Smoke `#999999` · Seam `#1E1E1E`

## Files

- `logos/` — mark, full logo (dark + light backgrounds), badge (any background), email logo (560px)
- `icons/` — app icon SVG + PNG at 16–1024, PWA maskable, 180 = apple-touch
- `ads/` — 4 square social ads (1080×1080), cover banner (1500×500), og-image (1200×630).
  SVG sources included — edit copy and re-export any time.
- `REVIEW-SHEET.png` — everything on one page (stale as of the colour revert — regenerate if needed)
