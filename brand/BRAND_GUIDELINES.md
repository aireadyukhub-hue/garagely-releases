# GarageLY — Brand Assets & Guidelines

## Colours
- **Accent / orange:** `#F4A523`  (buttons, "ly", hexagon, highlights)
- **Dark / background:** `#0f1117`
- **Near-black text:** `#1f2430`
- **White:** `#ffffff`

## Logo files (in this folder)
- **`garagely-logo-orange.png`** — primary logo. White "Garage" + orange "ly" + orange hexagon,
  transparent background. Use on **dark backgrounds** (the app's dark header, dark emails).
- **`garagely-badge-orange.png`** — badge version. The logo on a dark rounded panel with an
  orange outline. Self-contained, so it works on **any background** (light, grey, photos).
- **`garagely-logo-blue.png`** — original blue brand colour. Legacy; kept for reference.

## Usage
- Prefer the **orange** logo/badge — it matches the app's orange accent everywhere.
- On dark surfaces: use `garagely-logo-orange.png` directly.
- On light, grey, busy or unknown backgrounds: use `garagely-badge-orange.png`.
- Don't add extra outlines/shadows to the plain logo — use the badge instead when you need it
  to stand out on a light background.
- Keep clear space around the logo (roughly the height of the hexagon on all sides).

## Source
Vector source SVGs live in `../public/assets/garagely-logo-*.svg` (hexagon + "Garagely").
The orange versions are recoloured from `#3B82F6`/`#1F6FEB` → `#F4A523`.
